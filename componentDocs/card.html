    # card

    卡片组件

    # 示例:

    vue

    # 代码:

    <m-card imgSrc="/logo.png" summary="来康ui组件库示例summary" ></m-card>

    # Attributes:

    参数 说明 类型 是否必要 默认值
    width 卡片宽度 Number false -
    imgSrc 图片资源地址 String true -
    imgHeight 图片高度 Number false -
    summary 卡片概要 String\Slot false -
    footer 卡片底部 Slot false -
